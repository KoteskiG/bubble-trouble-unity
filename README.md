Flash game Bubble Trouble clone project in Unity.

# Play in browser
Don't let the bubbles hit you before you hit all of them, and watch out for the timer! Enjoy!

[ https://play.unity.com/en/games/b965a483-f1c7-4785-ad22-d3a02a4b3132/webbuild ]

# Controls:

- Move Left: A or left arrow key
- Move Right: D or right arrow key
- Shoot: Spacebar

# Built with
- Unity 2D
- C#
